  # LINEAL_MMC
  
  ## Descripción
  Programa en Python para cálculo de regresión lineal
  usando el Método de Mínimos Cuadrados (MMC).
  
  ## Requisitos
  - Python 3.14 o superior
  - Librerías: numpy, matplotlib, scipy
  
  ## Instalación
  1. Clonar el repositorio:
     git clone https://github.com/TU_USUARIO/LINEAL_MMC.git
  2. Crear entorno virtual: python -m venv venv
  3. Activar entorno: venv\Scripts\activate
  4. Instalar dependencias: pip install -r requirements.txt
  
  ## Ejecución
  python main.py
